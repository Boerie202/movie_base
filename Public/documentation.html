Movie Base



The purpose of Movie Base API

To build the server-side component of a “movie” web application. The web app will provide users with access to information about different movies, directors and genres. Users will be able to sign up, update their personal information and create a list of their favorite movies.



Request URL HTTP Method Response Examples
Get a list of movies /Movies GET Returns a JSON list of movies.
Return movie data by title about a single movie /movies/:title GET Returns Json data about a specific movie
Return movie data by name/title about a genre /movies/genre/:genreName GET Returns Json data about a genre and includes a decsription
Return data about a director by name /movies/director/:directorName GET Returns Json data about a director by using their name
Allow new users to register /users POST Allows to data to be pushed to the server and register new users, also assigns them a UUID
Allow users to update their username using their user ID /users/:id PUT Updates the Json data username by using a user's ID
Allow users to add a movie to their list of favourites /users/:id/:movieTitle POST Added a movie name to the user's Json data and send a message saying it's been completed
Allow users to remove a movie from their list of favourites /users/:id/:movieTitle DELETE Removes a movie from the list of favourites and send a message to say it's been done
Allow existing users to deregister themselves from the database /users/:id DELETE Checks by user ID to see if a user exists and removed them from the database if the result is true. Send message to say it's been completed
